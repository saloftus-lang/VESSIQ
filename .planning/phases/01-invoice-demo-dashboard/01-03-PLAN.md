    **JS: ROI calculator handler:**
    ```javascript
    document.getElementById('roiInput').addEventListener('input', function(e) {
      // Strip non-numeric characters
      const raw = e.target.value.replace(/[^0-9]/g, '');
      const amount = parseInt(raw, 10);

      const recoveryEl = document.getElementById('roiRecovery');
      const feeEl = document.getElementById('roiFee');
      const netEl = document.getElementById('roiNet');

      if (!amount || amount === 0) {
        recoveryEl.innerHTML = '$&mdash;';
        feeEl.innerHTML = '$&mdash;';
        netEl.innerHTML = '$&mdash;';
        return;
      }

      const annualRecovery = amount * 12 * 0.035;
      const vessiqFee = annualRecovery * 0.25;
      const netBenefit = annualRecovery - vessiqFee;

      const fmt = n => n.toLocaleString('en-US', { style: 'currency', currency: 'USD', maximumFractionDigits: 0 });

      recoveryEl.textContent = fmt(annualRecovery);
      feeEl.textContent = fmt(vessiqFee);
      netEl.textContent = fmt(netBenefit);

      // Format the input field itself with commas
      if (raw) {
        e.target.value = parseInt(raw, 10).toLocaleString('en-US');
      }
    });
    ```

    The ROI calculator uses `oninput` (via the `input` event), NOT a submit button. Outputs recalculate on every keystroke. Formula: annual recovery = input * 12 * 0.035, fee = recovery * 0.25, net = recovery - fee. If input empty or 0, show "$--" dash character.
