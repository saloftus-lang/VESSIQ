The Process

Dead simple by design.

We built VESSIQ so that getting to clean data takes minutes, not months.

01

Connect your sources

Point your EDI feed or terminal export at VESSIQ. We handle the parsing — no format configuration needed on your end.

02

We normalize everything

Every event maps to the same schema — regardless of where it came from. Vessel, voyage, container, port, timestamps, references.

03

Use the data

Query via REST API. Filter by vessel, port, event type, or time window. Build dashboards, triggers, and downstream feeds on top of it.
